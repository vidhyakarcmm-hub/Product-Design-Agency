## Product Design UI - designing agency
- A clean and modern landing page concept for a Product Design Agency.

  # Overview
  - Goal: Design a sleek and user-friendly landing page that effectively presents agency services and drives client interaction.
  - Tools: Figma, Adobe Photoshop
  - Duration: 4 weeks

  # features
  - Gradient background with smooth color blending
  - Modern navigation bar with active state styling
  - Balanced layout with professional imagery and clean spacing
